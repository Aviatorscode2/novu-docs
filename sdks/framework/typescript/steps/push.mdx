<RequestExample>
    ```typescript Push
    await step.push('push', async () => {
        return {
            subject: 'You received a message',
            body: 'A new post has been created',
        };
    });
    ```
</RequestExample>

## Push Output

<ParamField path="subject" type="string" required>
    The title or subject of the provider
</ParamField>

<ParamField path="body" type="string" required>
    The message to be sent to the push channel.
</ParamField>

## Push Result

The `Push` step does not return any result object.
