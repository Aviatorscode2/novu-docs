## Appearance Prop

`appearance` prop can be used to customise the Inbox. It has three main keys: `baseTheme`, `variables` and `elements`.

- `variables`: The variables object allows you to set global styling variables that apply to multiple elements within the inbox. This can include colors, border radius, and other common styles. You can find the list of supported variables [below](#list-of-available-variables).
- `elements`: The elements object allows you to define styles for specific elements within the inbox. Each key corresponds to an element, and the value is an object containing `style properties` for that element or you can also pass your `css classes`.
  See list of available elements [below](#list-of-available-elements).
- `baseTheme`: The baseTheme object allows you to define the base theme for the inbox. This can include variables and elements that apply to the entire inbox, as well as additional themes that can be switched between. It accepts an array of objects, each containing variables and elements. Each object will override the previous object's styles.
  This enables to override the custom themes imported from the `@novu/react` package.
  Currently, we only export the `dark` theme.

```jsx
import { Inbox } from '@novu/react';
const appearance = {
          variables: {
            borderRadius: '0.5rem',
            colorBackground: 'white',
            colorForeground: 'black',
            colorSecondary: 'gray',
            colorSecondaryForeground: 'white',
            colorNeutral: 'black',
          },
          baseTheme: [
            {
              elements: {
                bellContainer: {
                  colorBackground: 'white',
                  colorForeground: 'black',
                  borderRadius: '0.5rem',
                },
                bellIcon: {
                  colorBackground: 'white',
                  colorForeground: 'black',
                  borderRadius: '0.5rem',
                },
                bellDot: {
                  colorBackground: 'red',
                  colorForeground: 'white',
                  borderRadius: '0.5rem',
                },
              },
              variables: {
                borderRadius: '0.5rem',
                colorBackground: 'white',
                colorForeground: 'black',
                colorSecondary: 'gray',
                colorSecondaryForeground: 'white',
                colorNeutral: 'black',
                colorPrimary: 'blue',
                colorPrimaryForeground: 'white',
              },
            },
          ],
          elements: {
            back__button: {
              colorBackground: 'white',
              colorForeground: 'black',
              borderRadius: '0.5rem',
            },
            bellContainer: {
              colorBackground: 'white',
              colorForeground: 'black',
              borderRadius: '0.5rem',
            },
            // You can also pass your custom CSS classes
            bellIcon: 'custom-bell-icon p-4 bg-white rounded-full',
            bellDot: {
              colorBackground: 'red',
              colorForeground: 'white',
              borderRadius: '0.5rem',
            },
          },
        }

<Inbox
  appearance={appearance}
  options={{
    subscriberId: "SUBSCRIBER_ID",
    applicationIdentifier: "APPLICATION_IDENTIFIER",
  }}
/>;
```

### List of available variables

- `borderRadius`: The border radius of the elements.
- `colorBackground`: The background color of the elements.
- `colorForeground`: The foreground color of the elements.
- `colorSecondary`: The secondary color of the elements.
- `colorSecondaryForeground`: The secondary foreground color of the elements.
- `colorNeutral`: The neutral color of the elements.
- `colorPrimary`: The primary color of the elements.
- `colorPrimaryForeground`: The primary foreground color of the elements.
